## What's New

- Spotlight mode: select text anywhere, press ⌘⇧R to instantly refine it
- Modes: built-in CORRECT, TRANSLATE, and ASK modes with custom mode creation
- Flows: chain multiple modes together for multi-step text processing
- Quick Actions: assign global shortcuts to process text without opening Refine
- Local AI: download and run Qwen3 4B or Gemma 3 4B models privately on-device
- Cloud providers: connect OpenAI, Anthropic, or Ollama for cloud-based processing
- Clipboard history: access recent clipboard entries with ⌘⇧V
- Processing history: review all past refinements
- Customizable accent color and notification sounds
- Launch on login and automatic update checks
- Onboarding wizard for first-time setup
- Refine PRO licensing for premium features
