## Refine v0.1.8

### Highlights

- Updated Spotlight primary action wording from `Send` to `Refine` for better product clarity.
- Added clearer in-progress state: button now shows `Refining` with animated jumping dots during processing.

### Improvements

- Footer shortcut hint now uses `refine` instead of `send` to match the updated action naming.
- Loading label and animation are grouped in the button for better readability while processing.

### Behavior Change

- After `Paste`, Spotlight now closes automatically.
- Paste still targets the previous app correctly, but Refine no longer stays open after paste.
